# Security Policy

## Supported Versions

Currently, only the latest version of SWU OSR is being supported with security updates.

| Version | Supported          |
| ------- | ------------------ |
| Latest  | :white_check_mark: |
| Older   | :x:                |

## Reporting a Vulnerability

Please do not report security vulnerabilities through public GitHub issues.

If you believe you have found a security vulnerability in SWU OSR, please report it to us privately. You can do this by using GitHub's private vulnerability reporting feature if enabled, or by contacting the maintainers directly.

Please provide a detailed description of the vulnerability, including steps to reproduce it and any potential impact. We will make every effort to acknowledge your report within 48 hours and provide updates as we investigate and address the issue.
